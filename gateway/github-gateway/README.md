# github-gateway

> A GitHub App built with [Probot](https://github.com/probot/probot) that Webhook handler for GitHub events

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```

## GitHub Actions Environment Variables
* GITHUB_WORKFLOW
* GITHUB_ACTION
* GITHUB_ACTOR
* GITHUB_REPOSITORY
* GITHUB_EVENT_NAME
* GITHUB_EVENT_PATH
* GITHUB_WORKSPACE
* GITHUB_SHA
* GITHUB_REF
